# PokeRNG

โปรเจคสุ่มโปเกมอน (Random Pokémon Generator)  
เขียนด้วย HTML + CSS + JavaScript

## วิธีใช้งาน
เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย  
จะมีปุ่มให้กดสุ่มโปเกมอน พร้อมแสดงชื่อและรูปภาพ

## ฟีเจอร์
- สุ่มโปเกมอนแบบสุ่มทุกครั้ง
- แสดงรูป + ชื่อโปเกมอน
- UI เรียบง่าย
- มีระบบ Storage เก็บ Pokemon ที่ได้
- มีระบบ Auto Roll
- มีระบบ Auto Delete Pokemon
